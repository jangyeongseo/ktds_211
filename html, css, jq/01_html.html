HTML의 기본 구조

HTML 문서는 요소(Element)로 구성되어 있습니다. 요소는 태그(Tag)로 시작하여, 콘텐츠를 포함하고, 다시 태그로 닫힙니다.

                <태그>콘텐츠</태그>

예시:

                <h1>제목</h1>
                <p>문단 내용</p>
